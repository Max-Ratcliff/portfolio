# Max Ratcliff - Portfolio

This is my personal portfolio! It is a lightning-fast, static site built with **Astro** and **Tailwind CSS**. It serves as a simple way to view my math modeling, machine learning, and computer engineering projects, academic research, and blog posts.

## 🚀 Quick Setup

To run this project locally:

```bash
npm install
npm run dev
```

## 🛠️ Tech Stack
- **Framework:** Astro 5
- **Styling:** Tailwind CSS
- **Architecture:** Astro Content Collections (Markdown/MDX)

*(Originally based on the Astro Nano theme https://github.com/markhorn-dev/astro-nano). *
